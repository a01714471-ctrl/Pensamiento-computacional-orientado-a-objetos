# PROYECTO: Gestor Básico de Basketball
Este proyecto es un sistema básico para modelar elementos del basketball utilizando Programación Orientada a Objetos en C++.
El objetivo principal es practicar la creación y uso de clases, atributos, métodos y objetos, manteniendo un proyecto sencillo y fácil de extender.

En este sistema se manejan tres tipos de objetos:

Jugador
Entrenador
Equipo

Cada clase contiene su propio constructor, atributos privados, getters, setters y métodos específicos que representan acciones reales del basketball.

# Objetivo del Programa
El programa permite:

Crear jugadores con estadísticas básicas como puntos por juego.
Crear entrenadores con años de experiencia.
Crear equipos con su cantidad de campeonatos.
Mostrar información detallada de cada objeto.
Simular acciones:
  El jugador puede anotar puntos.
  El entrenador puede motivar al equipo.
  El equipo puede ganar partidos y campeonatos.

Este proyecto sirve como base para futuros avances donde se pueden agregar menús, archivos, herencia, lógica más compleja, etc.

# ¿Cómo se usa?
En esta versión inicial:

El archivo main.cpp contiene objetos de prueba creados manualmente.
Al ejecutar el programa, se muestran en consola los resultados de los métodos de cada objeto.
